# 元大可轉債查詢系統 v3.0

一個整合元大證券可轉債資料的網頁查詢系統,提供即時報價、初級市場、歷史價格等完整查詢功能。

## 🎉 v3.0 重大更新

### 改用單一資料檔案!
- **之前**: 需要3個Excel檔案
- **現在**: 只需要1個Excel檔案! ⭐

---

## ⚠️ 重要提醒

**此系統需要在本地HTTP伺服器運行**,不能直接雙擊index.html開啟!

原因:
- 瀏覽器的CORS安全限制
- file://協議無法載入Excel檔案  
- 必須使用http://協議

---

## 📦 必要檔案清單 (只需2個!)

1. ✅ **index.html** (72KB) - 查詢系統主程式
2. ✅ **00_CB代號名稱.xlsx** (1.1MB) - 完整資料庫

**就這麼簡單!** 🎉

## 🚀 快速開始

### 方法1: Python (推薦)

```bash
# 將兩個檔案放在同一資料夾,然後執行
python -m http.server 8000

# 開啟瀏覽器
http://localhost:8000
```

### 方法2: VS Code Live Server

1. 安裝「Live Server」擴充功能
2. 右鍵index.html → Open with Live Server

### 方法3: Node.js

```bash
npx http-server -p 8000
```

## 🎯 主要功能

### 1. 即時報價 💹
- 完整的可轉債報價資訊
- **掛牌最高/最低價** (歷史價格,2232筆)
- **高低振幅%** (價格波動範圍)
- 溢折價、波動率等指標

### 2. 初級市場 🆕
- **詢圈/競拍案件**: 進行中的發行案件 (28筆)
- **董事會公告**: 已通過但尚未發行的案件 (28筆)

### 3. 二級市場 📊
- **每日行情**: 完整每日行情資料 (339筆)
- **流通餘額**: 統計卡片 + 剛掛牌提醒 + 排序 (339筆)
- **贖回資訊**: 公司執行贖回權通知 (6筆)
- **停止轉換**: 停止轉換期間提醒 (14筆)
- **價格調整**: 轉換價格調整通知 (9筆)

### 4. 進階功能 🔧
- **報價篩選器**: 多條件進階篩選
- **多欄位排序**: 點擊表頭排序
- **瀏覽統計**: 今日/累積瀏覽量
