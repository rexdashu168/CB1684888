💻 部署到網路
-------------
參考 DEPLOY.md 檔案,三步驟完成:

1. 上傳到GitHub
   git init
   git add .
   git commit -m "初始化"
   git push

2. 啟用GitHub Pages
   Settings > Pages > Source: main

3. 完成!
   網址: https://你的用戶名.github.io/cb-tools/

📅 每週更新流程
---------------
方式一: 手動更新
1. 複製 template.json
2. 填入PDF週報資料
3. 命名為 YYYYMMDD-YYYYMMDD.json
4. 放到 data/weekly/
5. Git push

方式二: 使用Excel (未來功能)
1. 上傳Excel檔案
2. 自動解析
3. 自動更新
